🎤 Voice Weather App
A beginner-friendly voice-controlled weather application built using HTML, CSS, and JavaScript.
The app listens to the user's voice, fetches real-time weather data using the OpenWeatherMap API, changes the background and weather emoji dynamically, and even speaks the weather result aloud.

🚀 Features
🎤 Voice Recognition using Speech API
🌤 Real-time weather updates
🔊 Text-to-Speech weather response
🎨 Dynamic background changes based on weather
😎 Dynamic weather emojis
⏳ Loader animation while fetching data
📱 Responsive design for mobile and desktop

🛠 Technologies Used
HTML5
CSS3
JavaScript
OpenWeatherMap API
Speech Recognition API
Speech Synthesis API

🎯 How to Use
Click the 🎤 Speak button
Say:
Weather in Bangalore
The app will:
Detect your voice
Fetch weather data
Display weather details
Change emoji and background
Speak the result aloud

🌈 Weather Effects
Weather	Emoji	Background
